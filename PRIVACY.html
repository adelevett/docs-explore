AnswerDock Privacy Policy

Effective date: April 9, 2026

AnswerDock is a Chrome extension that helps users search for information and insert citations while working in Google Docs.

What data is processed

  • Search terms entered in the sidebar.
  • Selected text or link URLs when a user explicitly uses the context menu action.
  • Local preferences such as selected default search engine and always-open-search mode.

How data is used

  • Search terms are sent to DuckDuckGo Instant Answer API to retrieve instant answer results.
  • When users choose web search fallback or auto-open search mode, the extension opens the selected engine URL in a new tab.
  • Context menu text/link capture is used only to insert content into the user's open Google Docs tab.

Storage and retention

  • Preferences are stored in chrome.storage.local on the user's device.
  • The extension does not run a remote database or user account system.

Data sharing

The extension does not sell user data and does not share data with third parties except where required to provide functionality:

  • DuckDuckGo receives search terms for instant answer lookups.
  • Selected search engines receive normal query parameters when users open search in a new tab.

Permissions rationale

  • storage: save local settings.
  • contextMenus: allow right-click capture actions.
  • tabs: open and communicate with Google Docs tabs and search tabs.
  • scripting: inject sidebar code into Google Docs pages.
  • host permissions for docs.google.com and api.duckduckgo.com: required to operate inside Docs and fetch instant answers.

Your choices

  • You can disable or remove the extension at any time from chrome://extensions.
  • You can clear extension settings by removing the extension.

Contact

For support or privacy questions, open an issue in the public repository: github.com/adelevett/docs-explore/issues
